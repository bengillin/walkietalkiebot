Features

🎙
Voice Input
Talk to Claude using the Web Speech API. Wake word detection, tap-to-talk, and continuous listening modes.
🎨
6 Retro Themes
Every pixel reskins: TalkBoy, Bubble, Dial-Up, Finder, Guestbook, and 1984. Each with 70+ CSS custom properties.
💻
Claude Code Integration
Spawns Claude Code CLI with full tool access. Watch file edits, bash commands, and web searches in real-time.
📼
Cassette Tape UI
Conversations stored as cassette tapes with spinning reels, liner notes, and a tape collection browser.
📝
Plans & Liner Notes
Auto-detect plans in Claude responses. Add liner notes to any conversation for context and documentation.
🔍
Full-Text Search
FTS5-powered search across all conversations. Find any message instantly with Cmd+K.
🔗
MCP Server
30 MCP tools let Claude Code talk to Talkie. Create jobs, manage conversations, and bridge IPC.
✈️
Telegram Bot
Talk to Claude from Telegram. Voice messages, images, and conversation syncing across devices.
